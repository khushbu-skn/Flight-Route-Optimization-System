# ✈️ Flight Route Optimization System

A graph-based flight route optimization system developed in **C++** that helps users discover optimal flight paths between airports based on multiple search criteria. The project models airline networks as weighted graphs and employs efficient graph algorithms to compute the best routes while considering factors such as travel cost, flight duration, and layovers.

---

## 📌 Overview

The Flight Route Optimization System simulates a real-world airline network where airports are represented as graph vertices and flights as weighted directed edges. Users can search for routes using different optimization strategies and compare the results generated by multiple graph algorithms.

This project demonstrates practical applications of graph theory, shortest path algorithms, and object-oriented programming in solving transportation and network optimization problems.

---

## 🚀 Features

* Search the cheapest flight route between airports
* Find the shortest travel path
* Compute routes with minimum layovers
* Interactive airport and destination selection

## ⚙️ Algorithms

### Dijkstra's Algorithm

Computes the minimum-cost path between two airports using a priority queue.

**Time Complexity:** `O((V + E) log V)`

---

### Breadth First Search (BFS)

Determines the route requiring the minimum number of flight connections.

**Time Complexity:** `O(V + E)`

---

### A* Search

Uses heuristic-based search to accelerate route computation while maintaining optimality.
